## 2. 窗口行为 (Task 28)

**Webview 完整工作**(系统日志确认):

```text
[com.apple.WebKit:Loading] firstVisualLayout=0.004
                            firstMeaningfulPaint=0.054
                            domContentLoaded=0.038
                            loadEvent=0.003
                            subresourcesFinished=0.003
```

**`SkyLight: invalid display identifier`** 是 macOS 锁屏会话下的已知行为
(WindowServer 不分配 display ID 给非交互式会话的 GUI 进程)。
解锁后会话切换后即可看到 1280×820 窗口。

**Dock 重开 / 关窗保活 / cmd+Q 退出**逻辑在 `desktop/src-tauri/src/main.rs` 已实现:

- `WindowEvent::CloseRequested` → `api.prevent_close()` + `window.hide()`
- `RunEvent::Reopen` (Dock click) → `win.show()` + `set_focus()`
- `RunEvent::ExitRequested` (cmd+Q) → `runtime::shutdown_sidecar()`

**建议人工验证**:
1. 解锁后,启动 Nexus.app
2. 点 X → 窗口消失,Dock 图标保留
3. 点 Dock 图标 → 窗口回来
4. cmd+Q → 完整退出,port 30000 释放

```text
POST /api/sessions → {"id":"2d5fbde5-...","title":"E2E persist test",...}
POST /api/sessions/{id}/messages → {"id":"6ca1d776-...","content":"E2E ..."}
GET  /api/sessions/{id}/messages → [same message]
```

**`~/.nexus/` 目录**:

```text
nexus.db       18MB
sessions       460 (含 1 个本轮新建)
messages       922 (含 1 个本轮新建)
AGENTS.md      86B

logs/          (rolling)
```

**PRAGMA**:`db.py` 在连接建立时启用 `foreign_keys=ON / journal_mode=WAL / synchronous=NORMAL`,
缺列自动 `ALTER TABLE ADD COLUMN` (本轮新增 session 不需要手动迁移)。
