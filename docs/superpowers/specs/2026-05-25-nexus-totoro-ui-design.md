# Nexus 龙猫主题 UI 设计规范

**日期**: 2026-05-25
**状态**: 已选择方案 A
**主题**: 森林绿 + 暖木色

---

## 1. 设计方向

以宫崎骏《龙猫》为灵感，采用 **森林绿 + 暖木色** 主题：

- 龙猫森林为灵感，深绿色侧边栏配温暖米色主区域
- 苔绿色 (#4A7C59) 作为强调色
- 龙猫肚皮色 (#E8D5B7) 作为高亮色
- 圆润柔和的边角 (16px)
- 真实的龙猫 GIF 作为 mascot

## 3. 组件设计

### 3.1 侧边栏 (Sidebar)

- 宽度: 256px (w-64)
- 背景: 线性渐变 #2D5A3D → #1E3D2A
- Logo: "🌲 Nexus" 使用 serif 字体
- 底部: 龙猫 GIF (65x65) + "森林精灵" 标签
- Toggle 开关区域带毛玻璃效果

### 3.2 头部栏 (Header)

- 高度: 50px
- 背景: #F5EDE0
- 下边框: 1px solid #E8DCC8
- 左侧: "MiniMax-M2.7" 文字
- 右侧: 连接状态指示 (绿色圆点 + "已连接")

### 3.3 消息气泡

- 用户消息: 右对齐，背景 #E8D5B7，圆角 16px 4px 16px 16px
- AI 消息: 左对齐，背景 #F5EDE0，圆角 4px 16px 16px 16px
- 思考过程: 带左侧 3px #4A7C59 边框，浅绿色背景

### 3.4 输入区域

- 输入框: 圆角 24px，border #D4C4A8
- 发送按钮: 44x44px，圆形，背景 #4A7C59
